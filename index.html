Phone Menu

Start inside the phone menu, jump into the games shelf, and control everything with the keyboard.

    Global controls

    • Arrows navigate menus and move in games
    • Enter confirm, launch, or use the primary action
    • Space secondary action in games that need one
    • Escape / Backspace close screens or quit back to the menu
    • M mute or unmute at any time
    • R restart the current game
    • I reopen the game help overlay on demand

    Project note
